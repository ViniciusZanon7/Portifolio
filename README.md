# Portfólio – Vinicius Zanon

## Sobre o projeto

Este projeto consiste no desenvolvimento do meu portfólio pessoal como desenvolvedor, criado com o objetivo de apresentar minhas habilidades, tecnologias utilizadas e projetos desenvolvidos.

O site reúne informações profissionais, certificações, stack de desenvolvimento, projetos e formas de contato, funcionando como uma página de apresentação profissional na web.

O projeto foi desenvolvido utilizando HTML, CSS e Bootstrap, com foco em organização de conteúdo, responsividade e experiência visual.

Acesse o projeto online:
https://vinizanon.vercel.app/

## Objetivo

O objetivo deste projeto é:

- Criar uma presença profissional na web
- Apresentar projetos desenvolvidos durante os estudos
- Demonstrar conhecimentos em desenvolvimento frontend
- Servir como portfólio para recrutadores e oportunidades na área de tecnologia

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap
- Bootstrap Icons

O projeto também utiliza recursos externos como CDN do Bootstrap e componentes responsivos para construção da interface.

## Funcionalidades

O portfólio possui as seguintes seções:

Apresentação  
Breve introdução profissional com informações sobre formação e área de interesse.

Objetivo profissional  
Descrição das metas e objetivos na área de desenvolvimento de software.

Stack principal  
Tecnologias principais utilizadas no desenvolvimento backend.

Outras tecnologias  
Ferramentas e linguagens adicionais que fazem parte do aprendizado.

Atualmente estudando  
Lista de tecnologias e conceitos que estão sendo estudados atualmente.

Certificados  
Seção com exibição de certificados obtidos durante a formação.

Projetos  
Apresentação de projetos desenvolvidos, incluindo descrição e links para acesso ao código ou demonstração.

Atividade no GitHub  
Integração com estatísticas públicas do GitHub.

Contato  
Links diretos para redes profissionais e meios de contato.
